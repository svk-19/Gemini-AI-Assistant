# Gemini AI Assistant

A conversational AI chatbot built using Python, Streamlit, and Google Gemini API.

## Features

* Conversation Memory
* Chat History
* Multiple AI Modes

  * Career Mentor
  * Python Teacher
  * Technical Interviewer
* Download Chat
* Clear Chat
* Streamlit Web Interface

## Technologies Used

* Python
* Streamlit
* Google Gemini API

## Run Locally

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```
